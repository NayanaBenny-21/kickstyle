# kickstyle

A full-stack e-commerce web application built with Node.js, Express, MongoDB, and Handlebars.  
Supports admin management, user authentication, product browsing, cart, orders, and payment integration.

---
## Features

### User
- Browse products by category
- Search products
- Add products to cart and wishlist
- Checkout and place orders
- Payment integration with Razorpay
- User profile management
- OTP-based authentication
- Wallet system

### Admin
- Manage products, categories, coupons, and offers
- View and update orders
- Sales reports and analytics
- Dashboard with key metrics
- OTP-based admin login

### Utilities
- PDF and Excel generation for reports
- Referral code generation
- Stock management helpers
- Offer calculations
